# 🍽️ Zomato Dataset - Exploratory Data Analysis (EDA)

## 📌 About
This project performs Exploratory Data Analysis on the Zomato dataset to uncover insights about restaurants, ratings, pricing, and customer preferences.

## 📊 Key Insights
- Most restaurants fall under "Good" and "Average" rating categories
- Higher cost does not always guarantee higher ratings
- North Indian cuisine dominates the dataset
- Restaurants with online delivery tend to get higher engagement

## 🛠️ Libraries Used
- Pandas
- Matplotlib
- Seaborn

## 📂 Dataset
Zomato restaurant dataset with details like ratings, cost, cuisines, location, and services.

## 🚀 How to Run
1. Clone the repo
2. Install dependencies: `pip install pandas matplotlib seaborn`
3. Open `Zomato-EDA.ipynb` in Jupyter Notebook and run all cells
